Aneleira

Para medir o tamanho de anéis, é comum usar uma aneleira, que é um objeto composto por vários aros numerados de 1 a 35. Para utilizá-la, basta colocar o dedo em cada um dos aros até encontrar o tamanho ideal para o anel. É importante que o aro da aneleira esteja bem ajustado no dedo, pois assim é possível evitar erros de medição. Algumas pessoas tendem a escolher aros um pouco maiores do que o tamanho ideal quando experimentam uma aneleira. Para garantir que o anel fique confortável durante o uso, é recomendado que o aro correto da aneleira fique bem justo no dedo.
